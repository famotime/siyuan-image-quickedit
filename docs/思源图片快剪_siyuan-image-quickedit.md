# 思源图片快剪 siyuan-image-quickedit

在思源笔记文档中的图片点击右键，出现菜单，提供快捷操作：

- 基本信息：大小、分辨率、颜色深度、宽高比，例：230.13KB，1024×572×24 (1.79)
- 快捷操作菜单：

  - 转为webP格式
  - 压缩到50%
  - 压缩到30%
  - 压缩到10%
- 后面3个操作命令尝试将图片存储空间压缩到原图的相应比例，首先按用户拖拽后的图片宽度比例减小分辨率，然后尝试降低颜色数等手段降低图片大小，直至满足要求，操作过程中要有进度提示信息
- 执行操作后，不删除原图片，而是在原图片所在段落后插入一个新段落块，内容包括处理结果说明和压缩后图片；例如：“压缩后图片分辨率1024×572，尺寸减少50%；大小100KB，存储空间减少70%。”
- 对本文档图片批量操作快捷命令

  - 全部转为webP格式
  - 全部压缩到50%
  - 全部压缩到30%
  - 全部压缩到10%
- 插件配置中提供选项，用户可选择哪些命令注册到思源笔记文档菜单
